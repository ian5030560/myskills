# Quality Checklist — CS/AI/ML Papers

All common checklist items must also pass.

## Method
- [ ] `## 3. Method` divided into numbered subsections (`### 3.1`, `### 3.2`, ...)
- [ ] `### 3.2 Model Architecture` uses original paper figure
- [ ] Mathematical formulas in `$$...$$` display LaTeX
- [ ] Implementation details in Parameter/Value table
- [ ] Pseudocode provided for core algorithm

## Experiments
- [ ] `### 4.1 Setup` present with datasets and metrics
- [ ] `### 4.2 Main Results` is a Markdown table
- [ ] Best results **bolded** in main results table
- [ ] `### 4.3 Ablation Studies` present with component breakdown
- [ ] All images from `images/` referenced in correct subsection

## Related Work
- [ ] Comparison table present with at least Approach / Core Idea / Limitation columns
