# Quality Checklist — Survey / Review Papers

All common checklist items must also pass.

## Taxonomy
- [ ] `## 2. Taxonomy / Categorization` present
- [ ] Mermaid mindmap or timeline diagram included
- [ ] Taxonomy clearly explained

## Key Approaches
- [ ] Methods grouped into logical `###` categories
- [ ] Each method has Key idea / Advantage / Limitation
- [ ] Comparison table included when 3+ methods in same category

## Comparison
- [ ] Main comparison table with Method / Metrics / Year columns
- [ ] SOTA method explicitly identified
- [ ] Qualitative trade-off table present

## Open Challenges
- [ ] `## 5. Open Challenges & Future Directions` present
- [ ] Each challenge has description and difficulty explanation
- [ ] Future directions listed

## General
- [ ] Survey scope and review methodology stated in Introduction
- [ ] Number of reviewed papers mentioned
